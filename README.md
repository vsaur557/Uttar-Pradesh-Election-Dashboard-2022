## Overview

This Power BI dashboard provides an interactive analysis of election data for Uttar Pradesh. The dashboard helps users explore party performance, vote distribution, candidate statistics, and gender-wise voting trends through interactive visualizations.

## Dashboard Preview



## Features

- Interactive district-wise filtering
- Party-wise vote distribution analysis
- Total votes by party comparison
- Top 3 parties by votes
- Top 10 candidates by votes
- Gender-wise vote distribution
- Key performance indicators (KPIs)
- Candidate and party performance insights

## Key Metrics

- Total Parties: 5
- Total Districts: 10
- Total Votes: 206M
- Total Candidates: 2000
- Average Votes Per Candidate: 103.05K

## Tools & Technologies Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

## Dashboard Insights

- Comparison of votes received by different political parties.
- Identification of top-performing candidates based on vote count.
- Analysis of gender-wise voting distribution.
- District-level filtering for detailed exploration.
- Party-wise vote share and performance comparison.

## Disclaimer

**Important Note**

This project was created for learning, practice, and portfolio purposes only.

The dataset used in this dashboard is a sample/synthetic dataset and **does not represent the official Uttar Pradesh Assembly Election 2022 data**. The values, vote counts, candidate names, party statistics, and insights displayed in this dashboard are intended solely for demonstrating Power BI skills such as:

- Data Cleaning
- Data Transformation
- Data Modeling
- DAX Calculations
- Dashboard Design
- Data Visualization

Therefore, the results shown in this dashboard should not be considered actual election statistics or official election outcomes.
